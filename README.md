## Project Structure

```text
.
├── index.html
├── styles.css
├── script.js
├── README.md
└── assets/
   └── resume.pdf
```

## Run Locally

From the repo root:

```bash
python3 -m http.server 8080
```

Then open:

- http://localhost:8080

## Edit Content

- Main content and sections: `index.html`
- Styling/theme/layout: `styles.css`
- Scroll reveals + active nav behavior: `script.js`

## Deploy on Vercel (Recommended Settings)

When importing the GitHub repo into Vercel:

- **Root Directory**: `.`
- **Framework Preset**: `Other`
- **Build Command**: leave empty
- **Output Directory**: `.`
- **Install Command**: leave empty/default

This is a static website, so no build step is required.
